卒論コード
====

#  概要
コード・入出力データの保存リポジトリ 

## 説明
このリポジトリは2024年度卒業論文「RNAの発現パターンが真の系統樹を反映する事例の一般性の検証」にて使用されたコード・データの一部を置いたものです。   
このリポジトリは使用したコード群・入力データ群・出力データ群のディレクトリ構成になっています。
